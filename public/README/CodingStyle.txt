[參數規範]
基本規範
  * 常數    : 採全部大寫，字與字之間加入底線(_)分隔；常數於宣告之後便禁止修改
  * 一般變數: 首字大寫
  * 除 tmp 外所有變數須於出現區塊的起始處宣告

前綴規範
  * tmp 迴圈內使用暫時性變數，在其他範圍任意覆蓋或刪除皆不會影響正常運作之參數 EX: tmpI
  * us 來自使用者輸入的資料，禁止直接送交 API，必須經由指定處理函式前處理 EX: usUserName
  * g 全域變數 EX: gTime

後綴規範(選用)
  * _[型別] 如有需要，可於變數後方加入資料型別 EX: Width_int

[函式規範]
基本規範
  * 前綴字之後每個首字大寫
  * 運算式、比較式當中盡量加 ( )
  * if, for, while 中就算只有一行也盡量加 { }

前綴規範
  * fn 一般函式
  * ifn interface 指定函式
